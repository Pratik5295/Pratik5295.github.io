AR Shooter

Description

Explored Unity AR Core and AR Foundation functionalities to create a shooter game, where you select a playable character from a selection of 2 player characters with different stats. Survive the hords of enemies attacking you. There are two types of enemies in the game, and the game is structured to easily handle more. This game was developed for mobile devices and an apk is available to download: here.

Gameplay

Highlights

  • Integration with AR Core: Integrated Unity with ARCore & AR Foundation packages, to create an empty game world on a real world plane.
  • Combat: Simple combat mechanic between player and the enemies where the player uses gun to shoot, and the enemies get close and perform melee attacks.
  • Character Selection: Select from two different characters with different stats to play.
  • Different classes of Enemies: Use of inheritence to create different enemy classes.
  • Enemy Waves: A 3 wave spawner created using scriptable objects to handle different enemies per wave and per level.
