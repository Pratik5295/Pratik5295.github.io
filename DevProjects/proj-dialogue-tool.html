Custom Dialogue Tool

Description

Designed and developed a custom Unity Engine Editor tool to create dialogues and conversations to be played directly in the game. Each dialogue message comprises of Speaker Name, message, Unique Ids and Dialogue Options. Drag and drop connections via nodes or their option nodes to form a dialogue tree.

Video Demonstration
Features

  • Dialogue System + UI: A dialogue system branching different dialogue options.
  • Node Editor Create, Edit, Move and Delete nodes with ease. Allows easier setting of connections between nodes.
  • Scriptable Objects: Tool creates scriptable objects of each dialogue tree, giving user the ability to edit them with ease in the future without the tool.
  • JSON Support: Tool allows files to be created in JSON format as well, allowing the user ability to edit them without the game engine.
