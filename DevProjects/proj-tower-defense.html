Project Details:

This ongoing project entails the development of a 3D Tower Defense game utilizing Unity and C#. The game revolves around defending your base/town hall against waves of incoming enemies. In addition to traditional tower placement, players have the ability to control a playable character (hero) equipped with various abilities to defeat enemies. Currently, the game features only one type of enemy (melee). The Enemy AI is designed to locate the town hall and move towards it with the intention of destruction. They can switch targets, attacking the player hero character when incoming and reverting to targeting the town hall once the hero is defeated. This system is implemented using Unity Navmesh Agent, Pathfinding, and State Machines. The user interface (UI) comprises simple buttons for tower placement, offering a choice between two types: Single Target and Area Effect Towers. Additionally, the UI enables the selection of player abilities during gameplay. While the game is being developed primarily for the PC platform, it can be seamlessly adapted for mobile platforms.
